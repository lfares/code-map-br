# CodeMap Brasil — Prototype

This is a minimal React + Vite + Tailwind prototype for the "CodeMap Brasil" academic research landing page.

Quick start

1. Install dependencies:

```bash
npm install
```

2. Run dev server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
npm run start
```

Railway

Railway detects `package.json`. Use `npm run build` then `npm run start` (Railway provides `$PORT`).
